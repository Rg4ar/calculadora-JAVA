# Calculadora Matemática en Java

Este proyecto es una calculadora básica hecha en Java
Permite realizar operaciones matemáticas simples usando un menú en consola.


 Que hace el programa?

El programa permite:
 Sumar dos números
 Restar dos números
 Multiplicar dos números
 Dividir dos números
 Evitar la división entre cero

El usuario elige la operación y luego ingresa los números.


Cómo funciona?

1. Se muestra un menú en pantalla
2. El usuario elige una opción
3. El programa pide los números
4. Se muestra el resultado
5. El menú se repite hasta salir
